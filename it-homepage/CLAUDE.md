# LTS IT ソリューション サービスページ ルールブック

## プロジェクト概要

株式会社ライフタイムサポートのサービス紹介ページ群（静的 HTML/CSS/JS）。
GitHub Pages または静的ホスティングで公開する想定のランディングページ集。

## ファイル構成

```
it-homepage/
├── index.html              ← トップページ
├── it-support.html         ← IT サポートサービスページ
├── ai-seminar.html         ← 生成AIセミナーページ
├── ai-training.html        ← AI研修ページ
├── device-sales.html       ← デバイス販売ページ
├── web-marketing.html      ← Webマーケティングページ
├── styles.css              ← 共通スタイルシート
└── script.js               ← 共通スクリプト
```

## 技術スタック

- **言語:** HTML / CSS / JavaScript（フレームワークなし、純粋な静的ファイル）
- **特別な依存関係:** なし（npm install 不要）
- **ビルド:** 不要（そのままブラウザで開ける）

## 作業ルール

### デザイン変更
- `styles.css` に共通スタイルを書く。各 HTML ファイルにインラインスタイルを書かない
- 色・フォント等のブランドカラーは `styles.css` の CSS 変数で管理する

### コンテンツ変更
- 会社名・連絡先・サービス料金は変更頻度が高い。変更時は **全 HTML ファイルを横断して** 確認すること
- 龍竹一生の肩書・実績数値は `lts-knowledge` の内容と一致させること

### 本番デプロイ
- 変更後はブラウザで全ページを目視確認してから公開する
- デプロイ先・方法が未定の場合はユーザーに確認すること

## Google広告データの活用

### Google広告アカウント情報
- GA4と連携済み（プロパティID: 383839223）
- GA4データ内の「Paid Search」チャネルでGoogle広告経由トラフィックを分析可能

### 広告分析の主要指標

#### 1. 広告費用対効果（ROAS）
```
ROAS = 広告経由の売上 ÷ 広告費 × 100
目標: 300%以上（広告費の3倍以上の売上）
```

#### 2. 顧客獲得単価（CPA）
```
CPA = 広告費 ÷ コンバージョン数
目標: ¥10,000以下（サービスによって変動）
```

#### 3. クリック単価（CPC）
```
CPC = 広告費 ÷ クリック数
目標: ¥500以下（キーワードによって変動）
```

#### 4. コンバージョン率（CVR）
```
CVR = コンバージョン数 ÷ クリック数 × 100
目標: 3%以上
```

### 広告分析レポート形式

```
【期間】2026/3/1〜3/12
【広告費】¥XXX,XXX
【クリック数】XXX（CPC: ¥XXX）
【コンバージョン数】XX件（CVR: X.X%）
【CPA】¥X,XXX

【キャンペーン別実績】
| キャンペーン名 | 広告費 | CV数 | CPA | 評価 |
|-------------|-------|-----|-----|------|
| XXXキャンペーン | ¥XX,XXX | XX | ¥X,XXX | ⭕️優秀 |
| XXXキャンペーン | ¥XX,XXX | XX | ¥X,XXX | ⚠️改善必要 |

【課題】
- XXXキーワードのCPAが高い（¥XX,XXX）
- XXX広告のCTRが低い（X.X%）

【改善提案】
1. [優先度:高] CPAが高いキーワードの入札を下げる
2. [優先度:高] CTRが低い広告文をA/Bテスト
3. [優先度:中] 除外キーワードを追加して無駄クリックを削減

【予算配分提案】
- 優秀なキャンペーンに+XX%予算シフト
- 不調なキャンペーンは一時停止を検討
```

### データ収集方法

#### GA4から広告データを取得
```
1. GA4 MCPツールで「Paid Search」チャネルのデータを取得
2. utm_sourceが「google」のトラフィックを抽出
3. 期間比較で広告効果の推移を分析
```

#### スプレッドシートから予算データを取得
```bash
# 広告予算管理シートがある場合
gws sheets spreadsheets values get --params '{"spreadsheetId": "SHEET_ID", "range": "広告予算!A1:Z50"}'
```

### 重要な分析視点

- **時系列分析**: 曜日別・時間帯別のパフォーマンス差
- **デバイス別分析**: PC vs モバイルの広告効果差
- **地域別分析**: エリアごとのCPA・CVR差
- **競合分析**: オークション分析データから競合の動きを把握

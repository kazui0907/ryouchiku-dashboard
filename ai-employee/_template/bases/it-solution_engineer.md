ITソリューション エンジニア — 業務コンテキスト

主な業務は以下の通りです：

- システム開発・ソフトウェア開発
  → 業務システム、Webアプリ、API の設計・実装
- 業務自動化ソリューションの開発
  → n8n, GAS, Python による自動化ワークフロー構築
- Google Workspace 導入・設定・連携
  → gws-cli を活用した一括設定、API連携実装
- クライアント向けIT支援
  → 要件定義、システム提案、実装、サポート
- 既存システムの改善・デバッグ
  → パフォーマンス改善、バグ修正、機能追加

主要技術: n8n, GAS, Python, Node.js, Claude API, Google Workspace API

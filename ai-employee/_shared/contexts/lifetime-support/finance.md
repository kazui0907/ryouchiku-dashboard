# ライフタイムサポート 財務状況・業績概況

## 売上規模

### 成長推移

| 時期 | 売上規模 |
|------|---------|
| 創業時 | 数千万円 |
| 現在（LTS単体） | 約5億円 |
| 現在（グループ全体） | 約8〜9億円 |
| 2046年目標 | 100億円 |

### 成長の特徴

- 創業以来、右肩上がりで成長を継続
- グループ化により事業規模を拡大
- 新規事業による更なる成長を計画

## 利益構造

### 限界利益率

| 指標 | 数値 |
|------|------|
| **限界利益率** | 約40%前後で推移 |

### 投資方針

現在は将来を見据えた積極投資フェーズ:

| 投資分野 | 内容 |
|---------|------|
| **採用投資** | 人材確保・教育への先行投資 |
| **DX・システム投資** | AI・自動化システムの開発・導入 |
| **新規事業投資** | 不動産再販事業、IT事業への先行投資 |

### 利益の考え方

> **意図的な投資による一時的な利益圧迫を許容**

- 単年度の営業利益・経常利益は変動する
- 長期的には強固な財務基盤と高収益体質を目指す
- 一人当たり生産性の向上を重視

---

## 財務指標の方針

### 重視する指標

| 指標 | 方針 |
|------|------|
| **粗利益額（限界利益）** | 固定費増加に対応するため、最優先で確保 |
| **一人当たり生産性** | DX推進により継続的に向上させる |
| **限界利益率** | 40%前後を維持・向上 |

### 短期と長期のバランス

- **短期**: 単年度利益よりも成長投資を優先
- **長期**: 高収益体質・強固な財務基盤の構築

## 2046年に向けた財務目標

### 目標数値

| 項目 | 現在 | 2046年目標 |
|------|------|-----------|
| 売上高 | 約8〜9億円（グループ） | 100億円 |
| 従業員数 | - | 200名 |
| 上場 | 非上場 | 視野に入れる |

### 達成に向けた戦略

1. **既存事業の深耕** — 主力事業の着実な成長
2. **新規事業の確立** — 不動産再販・IT事業の黒字化
3. **M&A** — 戦略的なグループ拡大
4. **海外展開** — アセアン市場への進出

## 財務に関する意思決定原則

### 投資判断の基準

1. **ROI（投資対効果）** — 長期的な視点で評価
2. **戦略適合性** — ビジョン・パーパスとの整合性
3. **リスク許容度** — 失敗しても致命傷にならない範囲
4. **撤退基準** — 事前に明確化し、迅速に判断

### 資金配分の優先順位

1. 人材への投資（採用・教育）
2. DX・システムへの投資
3. 新規事業への投資
4. 設備・インフラへの投資

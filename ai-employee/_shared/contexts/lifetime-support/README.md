# 株式会社ライフタイムサポート コンテキスト

このフォルダには、ライフタイムサポートに関する業務判断・提案を行う際の基本情報が格納されています。

## 使用方法

AIエージェントは、ライフタイムサポートに関連するタスクを処理する際、このフォルダ内の情報を参照して判断・提案を行います。

---

## ファイル構成

| ファイル | 内容 | 参照場面 |
|---------|------|---------|
| `company.md` | 会社概要・企業理念・ビジョン・経営哲学 | 理念に沿った提案・判断時 |
| `business.md` | 事業内容・ビジネスモデル・新規事業 | サービス提案・事業戦略検討時 |
| `history.md` | 成長の軌跡・沿革・フェーズ | 会社紹介・背景理解時 |
| `challenges.md` | 経営課題・危機対応・克服の歴史 | 課題解決・改善提案時 |
| `finance.md` | 財務状況・業績・投資方針 | 経営判断・予算検討時 |
| `services.md` | ITソリューション事業のサービス詳細 | IT事業の提案・見積時 |
| `cases.md` | 導入事例・実績 | 提案・営業資料作成時 |
| `contacts.md` | 連絡先・担当者情報 | 問い合わせ対応時 |
| `policies.md` | 営業方針・価格戦略・対応ルール | 営業・価格決定時 |

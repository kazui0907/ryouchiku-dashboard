# ライフタイムサポート サービス一覧

## 主要サービス（5本柱）

### 1. AIパーソナルトレーニング

| 項目 | 内容 |
|------|------|
| **概要** | 42Tokyo卒業の現役社長が直接指導するAI活用トレーニング |
| **対象** | どんなにIT音痴でもOK |
| **形式** | 1回5時間の集中講座 |
| **ゴール** | AIを使って仕事ができるようになる |

**特徴:**
- マンツーマン or 少人数制
- 実際の業務を題材にした実践形式
- 受講後も質問対応サポート

---

### 2. IT内製化サポート

| 項目 | 内容 |
|------|------|
| **概要** | 専属エンジニアが「社内IT担当」として必要な時に必要な分だけサポート |
| **料金** | 月額15万円〜 |
| **補助金** | 活用可能（費用削減） |

**提供内容:**
- 社内システムの構築・運用
- 業務自動化（n8n等）
- ツール導入・設定サポート
- IT関連の相談対応

---

### 3. WEBマーケティングサポート

| 項目 | 内容 |
|------|------|
| **概要** | 自社実績に基づく「売れる」WEBマーケティング支援 |
| **実績** | 月間200件以上の問い合わせを獲得 |

**提供内容:**
- Google広告運用代行
- LLMO対策（LLM最適化）
- ランディングページ改善
- コンバージョン最適化

---

### 4. デバイス販売

| 項目 | 内容 |

### 5. 生成AI活用セミナー

| 項目 | 内容 |
|------|------|
| **概要** | PC初心者でも業務を劇的に効率化できるセミナー |
| **期間** | 3ヶ月 |
| **特徴** | プログラミング不要 |
| **費用** | 人材開発支援助成金活用で実質負担3.75万円〜 |

**カリキュラム:**

- 業務への活用方法

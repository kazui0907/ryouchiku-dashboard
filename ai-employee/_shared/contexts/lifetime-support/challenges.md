# ライフタイムサポート 経営課題と危機対応

## 過去の危機（克服済み）

### 第2フェーズで直面した課題

| 課題 | 詳細 |
|------|------|
| **属人化** | 一部幹部への業務集中、ノウハウが個人に依存 |
| **労働環境の悪化** | 「気合いと根性」による長時間労働の常態化 |
| **人材の定着難** | 新人の高い離職率、採用コストの増大 |
| **KKD経営** | 勘・経験・度胸に頼った意思決定 |

### 克服のアプローチ

1. **組織改革**
   - 新卒採用の開始
   - 動画マニュアルの整備
   - 評価制度の刷新

2. **DX推進**
   - AIシステムの内製化
   - 業務プロセスのデジタル化
   - データに基づく意思決定への転換

3. **成果**
   - 離職率の大幅低下
   - 働き方改革の実現
   - 埼玉DX大賞受賞

## 現在の外部環境課題

### 業界全体の構造的課題

| 課題 | 影響 | 対応方針 |
|------|------|---------|
| **建築業界の人手不足・高齢化** | 職人確保の困難化、人件費上昇 | 施工効率化、教育体制強化 |
| **新築着工件数の減少** | 主力事業の市場縮小 | 新規事業展開、中古再販事業 |
| **参入障壁の低さ** | 競合激化、価格競争圧力 | 高付加価値化、ブランディング |

### 創業者の姿勢

> **「ピンチはチャンス」**

市場縮小に対しても、新規事業や海外展開（アセアン等）へ果敢に挑戦する。

## 現在の内部課題

### 1. 売上の季節変動

**課題:** 閑散期（特定の季節）における売上の波

**対応策:**
- 閑散期向け施策の開発
- ストック型事業（メンテナンス・月額サービス）の強化
- グループ間での繁閑期の相互補完

### 2. グループシナジーの最大化

**課題:** グループ会社間（LTS、LA、NK）の資源共有とシナジーが不十分

**対応策:**
- 共通システム・インフラの整備
- 人材交流・ノウハウ共有の促進
- 仕入れ・外注の一元化

### 3. 新規事業の確立

**課題:** 新規事業（不動産再販、IT事業）の黒字化と差別化戦略の構築

**対応策:**
- 明確なKPI設定とPDCA
- 自社の強み（施工力・DXノウハウ）を活かした差別化
- 段階的な投資と検証

### 4. 固定費増加への対応

**課題:** 急拡大・投資増に伴う固定費の増加

**対応策:**
- 粗利益額（限界利益率）の確保を最優先
- 一人当たり生産性の継続的向上
- 投資対効果の厳格な管理

## 課題への基本姿勢

### 意思決定の原則

1. **現状維持の打破** — 変化を恐れず、常に改善を追求
2. **未来への投資** — 短期利益より長期的な企業価値を重視
3. **データドリブン** — KKD経営からの脱却、数値に基づく判断
4. **スピード重視** — 完璧を求めず、素早く実行・検証・改善

### リスクテイクの考え方

- 失敗を恐れない組織文化
- 小さく始めて素早く検証
- 撤退基準の明確化
- 学びを次に活かす仕組み

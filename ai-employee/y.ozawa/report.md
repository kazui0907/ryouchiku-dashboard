## 用語解説（中学生向け）

| 財務用語 | わかりやすい説明 |
|---------|----------------|
| 経常利益率 | 「会社が普通に仕事をして、どれだけ儲かったかの割合」。受取利息なども含めた、最も実態に近い儲けの指標 |
| 損益分岐点売上高 | 「赤字にならないために最低限必要な売上金額」。これを下回ると赤字になる |
| 限界利益 | 「売上から、直接かかったコスト（外注費など）を引いた後に残るお金」。固定費の支払いや利益のもとになる |
| 変動費（原価） | 「売上が増えると一緒に増えるコスト」。施工外注費などがここに入る |
| 固定費 | 「売上に関わらず毎月必ずかかるコスト」。人件費・広告費・減価償却費など |
| 労働分配率 | 「会社が稼いだお金のうち、人件費に使った割合」。高すぎると人件費負担が重い |
| 流動資産 | 「1年以内に現金化できる財産」。現金・売掛金など |
| 流動負債 | 「1年以内に支払わないといけない借金や未払い金」 |
| 粗利率 | 「売値から直接コストを引いた後の儲けの割合」 |
| フリーキャッシュフロー | 「最終的に会社が自由に使えるお金の増減」 |

## 経常利益率チェック（最優先確認）

10期（2025年度）計画の経常利益率は 3.9% — 辛うじて3%ラインをクリア。ただし極めて薄い余裕しかない。

過去期の経常利益率推移：

| 期 | 年度 | 経常利益率 | 判定 |
|----|------|----------|------|
| 5期 | 2022 | 18.3% | 優秀 |
| 6期 | 2023 | -3.9% | 死守ライン未達 |
| 7期 | 2023(半年) | 7.6% | 良好 |
| 8期 | 2024(一部) | -4.5% | 死守ライン未達 |
| 10期 | 2025(計画) | 3.9% | ギリギリクリア |

6期・8期と2度の赤字を経験しており、10期は計画値でギリギリ3.9%。月次KPIで問合せ件数・受注件数が計画を大幅に下回る月が続いており、年間計画の達成は不透明。実態では3%を割り込むリスクが高い。

## 詳細分析サマリー

### 強み（プラス評価）

1. **顧客単価が上昇傾向** — 1月実績170,663円は予測比68%と低いが、10月実績214,105円（目標比107%）など高単価を取れている月もある。顧客単価を引き上げる力がある
2. **受注率が高い月は55〜59%** — 業界標準と比較して高水準。問合せさえ増えれば受注は取れる構造
3. **手元キャッシュが厚い（6期末時点）** — 約2,187万円の現預金は月商比8ヶ月分。積極投資の安全網になっている
4. **7期（半年）で黒字回復** — 6期赤字後に外注費コントロールで7.6%の経常利益率を達成。立て直し能力あり

### 課題（マイナス評価）

1. **問合せ件数が計画を大幅に下回る月が多い** — 11月51.6%・12月47.1%と計画比半分以下の月がある。広告費9.6Mを投じているのに集客が安定しない
2. **口コミ件数が極めて少ない** — 12月3件（計画比27.3%）・1月2件（計画比40%）。口コミはコスト0の集客源なのに活用できていない
3. **受注率が1月に急低下（40.1%）** — 10月59.9%→12月43.5%→1月40.1%と右肩下がり。問合せの質の変化か、対応力の問題
4. **顧客単価が1月に大幅低下（170,663円）** — 目標250,000円の68%。セラミックコーティング（138,908円）などの低価格プランへの誘導が起きている可能性
5. **固定費の構造が重い** — 広告費9.6M＋人件費14.25Mの合計23.85Mが固定費の大半を占める。売上が下がっても固定費は変わらないため、損益分岐点が高い
6. **損益分岐点超過が繰り返されている** — 6期108.4%・8期109.7%と売上が損益分岐点を下回る状態が複数回。構造的な問題

## 改善アクション（優先順位順）

### 最優先（今すぐやること）

1. **受注率低下の原因を特定する**: 1月の受注率40.1%（目標50%）は10月の59.9%から大幅に低下している。「問合せの質が変わったのか（広告のターゲットがズレている）」「対応スピードが落ちているのか」「競合に負けているのか」を件別に分析する。受注率が1%改善するだけで月間受注件数が約0.5件増え、年間で約100万円の売上増につながる（単価20万円で計算）

2. **口コミ獲得の仕組みを自動化する**: 12月3件・1月2件と口コミがほぼ取れていない。施工完了後のお客様へ「口コミ依頼メッセージ（Googleマップ誘導のリンク付き）」をLINEまたはメールで自動送信する仕組みを今すぐ導入する。口コミが増えると広告費をかけなくても問合せが増える（コスト0の集客）。目標は月10件。口コミ10件増で問合せが月10〜20件増える見込み

3. **1月の顧客単価低下の原因を調べ、上位プランへの誘導スクリプトを作る**: 1月の平均単価170,663円は目標250,000円の68%。ペットHappyコート（172,700円）やセラミックコーティング（138,908円）などの低価格プランに集中している可能性がある。LDK以外の部屋（洗面・トイレなど）の追加施工提案や、上位プランへのアップセルトークを標準化する。単価を200,000円に戻すだけで、1月の売上は38万円増える計算（19件×3万円UP）

### 重要（1〜3ヶ月以内にやること）

4. **広告費9.6Mの費用対効果を月次で管理する**: 広告費を5期0円から8期9.6Mへ増やしたが、6期・8期と赤字を経験している。現在の広告費が「どの媒体」「どのキーワード」「どの地域」で問合せを生んでいるかを媒体別に可視化する。成果が出ていない広告を削減し、成果が出ている媒体に集中投下することで、同じ広告費でも問合せ件数を増やせる。まず媒体別の問合せ件数・受注件数・単価をGoogleスプレッドシートで毎月集計する仕組みを作る

5. **損益分岐点を下げるために固定費の見直しを行う**: 10期の固定費計2,898万円は売上目標7,610万円の38%を占める。現在の損益分岐点は高い水準にあり、売上が少し下振れするだけで赤字になる構造。広告費の一部をROIの高い施策（口コミ獲得・SEO・SNS）に切り替え、純粋な広告費支出を段階的に削減する計画を立てる。固定費を200万円削減できれば損益分岐点は約330万円下がる

6. **月次の経常利益率を毎月モニタリングする体制を作る**: 現在の経常利益率3.9%は死守ラインギリギリ。問合せ件数が1〜2ヶ月連続で計画比60%以下になった場合、年度の経常利益率が3%を割り込む可能性がある。毎月末に「売上 vs 固定費」を確認し、着地見込みが3%を下回りそうな場合は翌月の変動費（外注コスト）を絞る判断を即座にできる体制を整える

### 推奨（3〜6ヶ月以内にやること）

7. **人件費の増加ペースを売上成長に連動させる**: 人件費が4期461万円→9期1,425万円と3倍以上に増えている。売上も同様に伸びていれば問題ないが、10期の経常利益率3.9%という数字を見ると、人件費の増加が利益を圧迫している可能性がある。新たな採用・昇給は「その人件費増加分を超える売上増加が見込めるか」を事前に試算してから実行する

8. **キャッシュフロー計算書の作成を税理士に依頼する**: 現在の財務データには貸借対照表と損益計算書はあるが、キャッシュフロー計算書（実際のお金の出入り）がない。積極投資型経営では「利益は出ているのに現金がない」という状況が起きやすい。月次ベースで営業CF・投資CF・財務CFを把握できる体制を整えることで、キャッシュ切れのリスクを事前に察知できる

## 総括

NKサービスは「積極投資型経営」の方針のもと、広告費を大幅に増やしながら売上拡大を図っている。10期計画（7,610万円）は過去最大規模であり、その方向性は正しい。

ただし現時点で3つのリスクが重なっている。

- 問合せ件数の計画未達（11月51%・12月47%）
- 受注率の低下傾向（59%→40%）
- 顧客単価の低下（25万円目標→17万円実績）

この3つが同時に改善されないと、10期の経常利益率3.9%という計画値は達成できない。最優先課題は「問合せ数を増やすこと」ではなく「今来ている問合せを確実に受注し、単価を上げること」である。広告費をさらに増やすより、受注率と単価の改善に集中することで固定費を増やさずに売上を伸ばせる。

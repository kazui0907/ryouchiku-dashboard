# /analyze スキル — 財務データ自動分析

## このスキルの目的

NKサービスの公式財務スプレッドシートからデータを取得し、
`finance-expert` エージェントにキャッシュ中心の財務診断をさせ、結果を `report.md` に保存するまでを一括実行します。

---

## 📊 財務データの参照先（最重要）

**NKサービス公式財務スプレッドシート:**
```
URL: https://docs.google.com/spreadsheets/d/1WJCTlK6AYIVY8ssKmZQzheavM_efH0N2lb2mrR5caps/edit?usp=sharing
スプレッドシートID: 1WJCTlK6AYIVY8ssKmZQzheavM_efH0N2lb2mrR5caps
```

このスプレッドシートが**データ取得の最優先ソース**です。
ローカルファイルより先に必ずこのスプレッドシートを参照してください。

---

## 実行手順（この順番を必ず守ること）

### STEP 1: 公式スプレッドシートからデータを取得する

まずシート一覧を取得する：

```bash
gws sheets spreadsheets get \
  --params '{"spreadsheetId": "1WJCTlK6AYIVY8ssKmZQzheavM_efH0N2lb2mrR5caps"}' \
  --fields "sheets(properties(sheetId,title))"
```

取得できたシート一覧を表示し、**全シートのデータを順番に取得する：**

```bash
gws sheets spreadsheets values get \
  --params '{"spreadsheetId": "1WJCTlK6AYIVY8ssKmZQzheavM_efH0N2lb2mrR5caps", "range": "シート名!A1:Z300"}'
```

**スプレッドシートの取得に失敗した場合（認証エラー・接続エラーなど）:**

```
⚠️ スプレッドシートへの接続に失敗しました。
ローカルファイルを代わりに検索します...
```
→ STEP 1b（ローカルファイル検索）に進む。

---

### STEP 1b: ローカルファイルを探す（スプレッドシート取得失敗時のみ）

`D:/scripts/ai-employee/y.ozawa/zaimu/` フォルダを優先して以下のパターンで検索する：

```
検索パターン（順番に実行）:
1. zaimu/**/*.csv
2. zaimu/**/*.xlsx
3. zaimu/**/*.xls
4. zaimu/**/*.pdf
5. zaimu/**/*.tsv
```

**ローカルファイルも見つからなかった場合:**
```
📂 財務データが見つかりませんでした。

以下のいずれかの方法でデータを用意してください：
  【方法1】スプレッドシートへのアクセス権限を確認する
           → gws auth status で認証状態を確認
  【方法2】ファイルを zaimu フォルダに配置する
           → D:/scripts/ai-employee/y.ozawa/zaimu/ に置いてください
           → 対応形式: CSV / Excel (.xlsx / .xls) / PDF / TSV
```
→ ここでスキルを終了する。


### STEP 3: finance-expert エージェントに分析を依頼する

2Dゲーム風ビジュアルを表示しながら、Agent ツールで `finance-expert` を呼び出す。

**ビジュアル表示例:**
```
╔══════════════════════════════════════════════════════════════╗
║  🎮 NKサービス AI社員システム — 財務分析モード              ║
╚══════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────┐
│ 🏢 オフィスマップ                                            │
│                                                              │
│    [社長室]          [財務部]                               │
│      🤖  ═══════►     💰                                    │
│                        ↓                                     │
│                     [分析中...]                             │

```

**finance-expert への指示文（Agent ツールの prompt に含めること）:**

```
以下の財務データを分析してください。

【データ取得元】
Googleスプレッドシート: https://docs.google.com/spreadsheets/d/1WJCTlK6AYIVY8ssKmZQzheavM_efH0N2lb2mrR5caps/edit?usp=sharing
（または ローカルファイル: ファイル名）

【データ内容】
（取得したスプレッドシート全シートの内容をすべてここに貼り付ける）

【分析要件】
1. 専門用語は必ず中学生でもわかる言葉に言い換えて説明すること
2. キャッシュを最優先に、NKサービス方式のスコアリング（キャッシュ60点・利益30点・参考10点）で採点すること
3. 経常利益率が3%未満なら冒頭に🚨警告を出すこと
4. スコアが79点以下の軸には具体的な改善アクションを優先順位付きで3つ以上提示すること
5. 分析完了後、結果を report.md に自動保存すること
```

## エラーハンドリング

| 状況 | 対応 |
|------|------|
| ファイルが空 | 「データが空です。内容を確認してください」と報告してスキップ |
| 文字化け | 「文字コードの問題で読み取れません。UTF-8で保存し直してください」と案内 |
| データ形式不明 | 読み取れた範囲で分析し、「一部データが不明な形式のため除外しました」と明記 |
| finance-expert が起動しない | エラー内容を表示し、「`.claude/agents/finance-expert.md` が存在するか確認してください」と案内 |
